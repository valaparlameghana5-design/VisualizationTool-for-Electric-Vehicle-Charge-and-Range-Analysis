# ⚡ EV Charge & Range Analytics Dashboard

### 🚗 Transforming Electric Vehicle Data into Powerful Insights

An interactive analytics platform that visualizes electric vehicle (EV) charging infrastructure, driving range, pricing, efficiency, and market trends. The project combines **MySQL**, **SQL**, **Tableau**, and **Flask** to deliver an engaging web-based dashboard for data-driven decision-making.

---

## ✨ Key Highlights

* 🔋 EV Charging Infrastructure Analysis
* 🚘 Brand & Model Comparison
* 💰 EV Price Analysis
* 📍 Interactive Charging Station Maps
* ⚡ Battery Range & Efficiency Insights
* 🚙 Body Style & Powertrain Distribution
* 📊 Interactive Tableau Dashboard & Story
* 🌐 Flask Web Application Integration
